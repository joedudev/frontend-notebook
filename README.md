### ğŸ“‹ Organization Principles

**Single Files** for topics that can be studied in one session:

- Small, focused topics
- Related concepts that flow together
- Quick reference materials

**Separate Folders** for:

- Complete roadmaps (HTML, CSS, JS, etc.)
- Complex topics with multiple subtopics (like Forms)
- Topics requiring extensive examples and practice

**Naming Convention**:

- Main roadmap files: `01-topic.html`
- Sub-roadmap files: `02.01-subtopic.html`
- Folders: `02-html-roadmap/`, `02.08-forms/`
